What is XML?

  • XML stands for EXtensible Markup Languaje
  • XML is a markup languaje much like HTML
  • XML was designed to describe data
  • XML tags are not predefined. You must define your own tags
  • XML uses a Document Type Definition(DTD) or an XML Schema to describe the data
  • XML with a DTD or XML Schema is designed to be self-descriptive

    • The main difference between XML and HTML

    XML was designed to carry data.

    XML is not a replacement for HTML

    XML and HTML were designed with different goals:

    XML was designed to describe data and to focus on what data is.

    HTML was designed to display data and to focus on how data looks.

    HTML is about displaying information, while XMLS is about describing information.

    XML does not Do anything

    Maybe it is a little hard to understan, but XML does not DO anything. XML was created to structure, store and to send information.
